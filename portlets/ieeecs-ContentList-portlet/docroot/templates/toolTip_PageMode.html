• View in New Page - The content (article, blog, webinar, etc) will be displayed in a new window.
    • This mode is the normal method you will see on most web sites.
    • A typical HTTP Request/Response, and browser refresh is seen.
  • View in Current Page - The content (article, blog, webinar, etc) will be displayed in the existing window.
    • This mode is a newer method, using AJAX to populate a "target" container.
    • For this reason, a "target" Container ID must be provided in this Content List portlet.
    • The target will be a Portlet of type "Content Container".
