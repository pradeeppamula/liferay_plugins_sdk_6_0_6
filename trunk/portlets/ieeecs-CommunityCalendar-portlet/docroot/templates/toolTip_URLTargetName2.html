• If you wanted something like
    • http://computingnow.computer.org/portal/web/computingnow/cloud/webinars
  • Then you would set the value of "URL Target Name" to "webinar".
  • If you wanted something like
    • http://computingnow.computer.org/portal/web/computingnow/cloud/booya/interviews
  • Then you would set the value of "URL Target Name" to "booya/interviews".
