• For every Content Entry item displayed in this Content List, the portlet will need to determine:
    • Community (Group) Name
    • Channel (Category) Name
    • Page Name
  • This will result in a URL for each link which will look like:
    • http://<>/<>/community-name/channel-name/page-name
    • For example:
    • http://computingnow.computer.org/portal/web/computingnow/cloud/content
    • Where:
    • Community (Group) Name = computingnow
    • Channel (Category) Name = cloud
    • Page Name = content
