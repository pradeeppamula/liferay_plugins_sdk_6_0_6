• This should be a unique name, for this portlet only.
    • Please keep this name short and alphanumeric only.
    • Please don't use any special characters or spaces.
  • This "target" name can be used to bind a "Content List" portlet to this Content Container Portlet.
  • Leaving the "target" name blank, or not using it in a "Content List" portlet will mean:
    • Clicking on a "Content List" link will require a request/response via the normal HTTP method.
    • The user will see a browser refresh/render of the new page.
  • Filling out the "target" name and then using it in a "Content List" portlet will mean:
    • Clicking on a "Content List" link will use an AJAX call to the server to get the content.
    • The user will *not* see a browser refresh/render of the new page.
    • Instead, the previous content will fade out, and then fade in with the new content.
    • There will be no visible page refresh/render.
